# Sample Images
This directory contains sample images for testing the Chromasonic pipeline.

## Usage
Place your test images here to experiment with the image-to-music conversion.

Supported formats:
- JPEG (.jpg, .jpeg)
- PNG (.png)
- GIF (.gif)
- BMP (.bmp)
- TIFF (.tiff)

## Recommendations
For best results:
- Use images with distinct, vibrant colors
- Avoid very dark or very light images
- Landscape and nature photos often work well
- Abstract art and colorful patterns create interesting melodies
